# Multi-Currency Expense Splitter

## Project Description

The Multi-Currency Expense Splitter is a web application that helps users manage and split expenses within a group, even when transactions are made in different currencies. It automatically converts currencies, calculates balances, and simplifies settlements.

---

## Features

* User Authentication (Login/Register)
* Group Creation & Management
* Add & Split Expenses
* Multi-Currency Support
* Automatic Currency Conversion
* Real-Time Balance Calculation
* Settlement System
* Dashboard Summary

---

## Technologies Used

* Frontend: React.js (Vite + TypeScript)
* Backend: Node.js, Express.js
* Database: MongoDB
* API: Currency Exchange API
* Tools: GitHub, VS Code, Postman

---

## Installation Steps

### 1. Clone Repository

git clone https://github.com/Jahnavipothapu/multi-currency-expense-splitter.git

### 2. Backend Setup

cd multi_currency_expense_splitter_final/backend
npm install
node server.js

### 3. Frontend Setup

cd ../frontend
npm install
npm run dev
